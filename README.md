# Expense-Tracker
Add, delete and edit your transactions. Create your own categories. Change app theme and accent color.

## Used Technologies
  - Swift
  - CoreData
  - Keychain
  - Localization
  - Custom UIAppearance
  - UIView Animation using animate & transition
  - Programmatic UI
  - Multiple Storyboards using Storyboard Reference
